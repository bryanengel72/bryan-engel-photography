YOUR PORTFOLIO PHOTOS LIVE HERE
================================

The website now AUTO-LOADS every image in this folder — any filename, any
number. Just drop photos in and they appear in the gallery.

ORDER:  Images show in name order. The FIRST one is also the homepage hero +
        social-share preview. The current files are named photo-01.jpg,
        photo-02.jpg, ... so photo-01 is the hero — rename to reorder.

TO ADD a photo:     drop a .jpg/.png/.webp in this folder. Done.
TO REMOVE a photo:  delete it from this folder. Done.
TO REORDER:         rename so they sort the way you want (01, 02, 03 …).

SIZE:  Keep images web-sized (long edge ~2400px, a few hundred KB to ~1MB).
       The current photos were already optimized down from the full-res files.
       Don't drop 15MB camera originals straight in — they slow the site.

CUSTOM ALT TEXT / CAPTIONS (optional, good for SEO):
       Edit the OVERRIDES map in  lib/getPhotos.ts , keyed by filename, e.g.
         "photo-01.jpg": { alt: "Jill, bikini competitor, golden hour", caption: "Jill" }

LOGO:  drop a transparent PNG at  public/logo.png  (one level up) — it appears
       in the navbar automatically.
